# 📚 Student Grade Tracker

## 📌 Description
Student Grade Tracker is a Java console-based application developed as part of the CodeAlpha Java Programming Internship. It helps manage student records and perform various grade-related operations.

## ✨ Features
- ➕ Add Student
- 📋 Display All Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Marks
- 🗑️ Delete Student
- 📊 Calculate Average Marks
- 🏆 Display Highest Marks (Topper)
- 📉 Display Lowest Marks
- 📑 Generate Summary Report

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Iterator
- IntelliJ IDEA

## 📂 Project Structure
```
src/
├── Main.java
├── Student.java
└── GradeTracker.java
```

## ▶️ How to Run
1. Clone this repository.
2. Open the project in IntelliJ IDEA.
3. Run `Main.java`.
4. Use the menu to perform different student management operations.
